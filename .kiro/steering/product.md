# Product Overview

The Bajaj Document Processing System is an AI-powered document processing platform that automates the validation and approval workflow for business documents (Purchase Orders, Invoices, Cost Summaries, and supporting materials).

## Core Purpose

Streamline document submission, validation, and approval processes for Bajaj Auto Limited using multi-agent AI architecture with Azure services.

## User Roles

- **Agency Users**: Submit document packages and track processing status
- **ASM (Area Sales Manager)**: Review submissions with AI recommendations and approve/reject packages
- **HQ (Headquarters)**: Access analytics dashboards, KPIs, and conversational AI insights

## Key Capabilities

- Automated document classification and data extraction using Azure Document Intelligence and GPT-4 Vision
- Cross-document validation with SAP integration
- Weighted confidence scoring (PO 30%, Invoice 30%, Cost Summary 20%, Activity 10%, Photos 10%)
- AI-generated approval recommendations with evidence
- Email notifications via Azure Communication Services
- Analytics with AI-generated narratives
- Conversational AI chat assistant with semantic search
- Role-based access control with JWT authentication

## Technology Approach

Multi-agent AI system orchestrated by Semantic Kernel, with specialized agents for document processing, validation, confidence scoring, recommendations, notifications, analytics, and chat.
